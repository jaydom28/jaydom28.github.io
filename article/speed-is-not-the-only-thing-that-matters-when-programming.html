Speed Is Not The Only Thing That Matters When Programming

Speed of the programming language itself shouldn't be the only metric used to measure how good the language is. C++ is a pretty fast language whereas python is a pretty slow language but a professional python programmer can probably write a program that runs faster than an identical c++ program written by an okay c++ programmer. Other things to consider are the speed of development and the expressiveness + readability of the syntax.
